Main Processing Overview

Remove old log file and initialize logger

Get time of day

Use regex to remove new line character from date and time string.

  • Print startup message with a list of arguments passed

validate arguments:

  1. Filename of the pcap trace. Make sure it exists and is in a pcap format

Open passed pcap file for reading packets

process filter if supplied ip:x.x.x.x - filter on IPaddress hp:x.x.x.x-y.y.y.y - filter oon a host pair port:p - filter on TCP port socket: x.x.x.x:srcport-y.y.y.y:dstport mac:hh-hh-hh-hh-hh-hh prot:tcp|udp... bpf: BOF string - use any of the BPF commands. Put filter at end of command line or enclose in quotes

Loop over file reading a packet, sending it to the parser, until EOF
